GOGUFW 0.5.11 call tone stable rollback/refinement

Changes in this package:
- Removed CllVol from the visible radio menu.
- Kept the reserved call_vol byte in Messenger config only for storage/layout compatibility.
- Kept CllTon menu below SetScn and above MsgRx.
- CllTon preview is now non-blocking and about 2 seconds long.
- MENU/EXIT/LEFT/RIGHT should interrupt the current preview.
- F+9 still sends the selected call tone over RF for about 3-4 seconds.
- CALLTX display text is retained.
- CALLTX local sidetone is reduced but not intentionally disabled.
- RF volume/gain experiments that caused robotic audio were removed.

Test focus:
1) CllTon preview should not freeze menu navigation.
2) CllTon value should update on screen immediately.
3) MENU should save immediately even during preview.
4) EXIT should leave immediately even during preview.
5) F+9 should still send clean call tone.
6) Local speaker during CALLTX should be quieter than previous build but still audible.
